# Themes

The 'Widgetsy' library provides five different themes that you can use to customize the appearance of the widgets. Each theme offers a unique color palette and design style, allowing you to choose the one that best suits your project's aesthetics. Here are the five available themes:

## GrapeFruit Sunset

The vibrant hue of orange envelops the room, casting a warm and inviting atmosphere that instantly captures attention.

export const Theme1 = () => (
  <div
    style={{
      display: "block",
      background: "linear-gradient(90deg, #e96443 0%, #904e95 100%)",
      height: "10vh",
    }}
  ></div>
);

<Theme1></Theme1>

```bash
background: "linear-gradient(90deg, #e96443 0%, #904e95 100%)"
```
